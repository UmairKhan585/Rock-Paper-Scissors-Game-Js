# Rock Paper Scissors Game

## Description

This is a simple Rock Paper Scissors game implemented using JavaScript, HTML, and CSS. The game allows you to play against the computer and keeps track of scores, providing a visually engaging experience with rock, paper, and scissors icons.

## Features

- **Play Against the Computer**: Choose rock, paper, or scissors and play against a computer opponent.
- **Score Tracking**: Keeps track of both your score and the computer's score.
- **Visually Appealing**: Designed with rock, paper, and scissors icons for an engaging user experience.

## How to Play

1. **Choose Your Move**:
   - Click on one of the icons: Rock, Paper, or Scissors.
   
2. **Computer's Move**:
   - The computer will randomly select its move.
   
3. **Determine the Winner**:
   - The game will display the winner and the reason for winning.
   
4. **Score Update**:
   - Your score and the computer's score will be updated accordingly.

## Code Structure

- **`app.js`**: Contains the game logic. It handles generating computer choices, determining the winner, and updating scores.
- **`index.html`**: The main HTML structure of the game, which sets up the game interface.
- **`mystyle.css`**: Styles the game elements to provide a visually appealing experience.

## Dependencies

- **No External Libraries Required**: This game is built using plain JavaScript, HTML, and CSS, with no additional libraries.

## Installation and Running

1. **Clone the Repository**:
   ```bash
   git clone <repository_url>
   ```

2. **Navigate to the Project Directory**:
   ```bash
   cd <project_directory>
   ```

3. **Open `index.html`**:
   - Open the `index.html` file in your web browser to start playing.

Have fun playing the Rock Paper Scissors game and challenge the computer!
